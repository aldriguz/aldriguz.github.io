# Laravel
Estos son los comandos que debemos de conocer al trabajar con Laravel.

## Ejecutar migraciones
```
php artisan migrate:install
```

## Inyectar data 
```
php artisan db:seed
```

## Crear API KEY
```
php artisan key:generate
```
